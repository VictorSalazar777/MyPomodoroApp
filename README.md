# MyPomodoroApp

App that shows a Pomodoro counter, 15 min or 20 min. I haven't 
added a break time counter yet. I will improve it in a future 
version.

I used:

* Android framework.
* Java.
* Audio.
* TDD.
* Mockito.
* Espresso.
* ViewModel (MVVM).
* Activity.
* Service.
* Broadcast receiver.
* Shared preferences.
* Material components.
* View binding.
* Custom dialog.
